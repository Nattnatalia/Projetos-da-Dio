Ferramentas Utilizadas

Para o desenvolvimento de uma pagina web existem algumas ferramentas que nos auxiliam nisso, uma delas é o Visual Studio Code é nesse programa que o texto é escrito e editado, existem outros programas que tem a função de desenvolver texto, o proprio documento de texto do computador pode ser utilizado nesse caso, porém através do Visual Studio Code isso é mais pratico o que facilita a vida do profissional, pois dentro do programa exitem recursos que podem ser baixados como por exemplo a extenção Live Server que serve para visualizar o HTML que esta sendo criado conforme vai ganhando forma.

Inspetor de Elementos

voltar

Esse recurso é utilizado para verificar como a pagina foi criada, quais ferramenas do html ou css (caso esteja sendo utilizado) correspondem a cada parte do texto e assim por diante. Para isso basta apertar F12 no teclado e pronto ja pode ser visualizado, outra forma é ir no canto superior direito da tela clicar nos três pontinhos > Mais ferramentas > Ferramentas do desemvolvedor.

Estrutura básica do HTML

voltar

Para que um HTML seja criado deve-se seguir uma estrutira básica veja:

Na imagem apresentada podemos ver tres tags chaves.
html: Essa engloba todo o documento do começo ao fim, é como se fosse a tag mãe, todas as outras estaram dentro dela.
head: É como se fosse um cabeçalho do desenvolvedor, ou seja, apenas ele pode vê-la ela fica oculta para o usuário.
body: Na tag body contem tudo que é exibido para o usuário, nessa esta o corpo do texto.

Falando sobre Tags

voltar

Além das tags estruturas citadas anteriormente existe uma infinidade de outras tags que podemos utilizar, dentre elas temos:

br: Utilizada para quebra de texto.
i: Coloca o texto em italico.
strong: Coloca o texto em negrito.
title: Utilizada para incluir um titulo.
input: Adiciona uma campo fechado para escrita, normalmente utilizado na criação de formulário.
img: Utilizado para adicionar uma imagem.

Atributos das Tags

voltar

Agora iremos falar sobre atributos. Essa função do html consiste em complementar uma tag, um atributo pode ser obrigatório ou apenas para se detalhar algo ou até mesmo ajudar o profissional a ligar um assusto a outro dentro do codigo. Alguns deles são:
id: Serve para identificar um elemento.
class: Utilizado para classificar uma tag.
alt: Detalha o que esta sendo mostrado na imagem fixada, um testo alternativo caso por algum motivo a imagem fixada no html não possa ser visualizada.
href: especifica a URL da página.
target: Abre o link adiconado em uma nova aba.

Textos

Aqui serão apresentadas algumas tags que podem ser utilizadas no meio do txto para personaliza-lo.
blockquote: Para adicionar uma citação.
title: Fazer uma interação como por exemplo, "clique aqui" quando se passa o mouse em cima de um link.
mark: Para marca texto.
sup:Adicionar ao superior ao texto como se fosse uma elevação.

Listas Ordenadas e Não Ordenadas

voltar

Uma lista ordenada as olé aquela que segue uma ordem definida por exemplo numerica, já uma lista não ordenada ul não é definida por numeros ou letras ela apenas pode ser pontuada. Não aponta necessariamente uma logica.

Links

Os links são apontados dentro de uma documento html atraves da tag a que vem de ancorar seguida do atributo href que leva o usuário para outra página fazendo assim a navegação possível dentro do código.
